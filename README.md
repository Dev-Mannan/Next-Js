<h2>🛠️ Installation Steps:</h2>

<p>1. Enter this command in your Directory terminal</p>

```
git clone https://github.com/Dev-Mannan/Next-Js.git
```

<p>2. after installation enter command</p>

```
cd Next-js
```

<p>3. final command</p>

```
npm run dev 
```
